##  🔧 ONE-TIME FIX NEEDED

**Issue:** GuidanceSection component uses `children` not `content` prop

**Affected Files:** 19 tools in developer/ and calculators/ folders

**Quick PowerShell Fix Script:**
```powershell
# Navigate to project root
cd "d:\Prasad\ILoveJPG"

# Create backup
Copy-Item -Path "src\pages" -Destination "src\pages_backup" -Recurse

# Fix all GuidanceSection in developer tools
$files = Get-ChildItem -Path "src\pages\developer\*.tsx" -File
foreach ($file in $files) {
    $content = Get-Content $file.FullName -Raw
    # Simple replacement: remove content prop, add children
    $content = $content -replace '<GuidanceSection title="([^"]+)" content=\{`([^`]+)`\} \/>', '<GuidanceSection title="$1"><p>$2</p></GuidanceSection>'
    Set-Content -Path $file.FullName -Value $content -NoNewline
}

# Fix all GuidanceSection in calculator tools  
$files = Get-ChildItem -Path "src\pages\calculators\*.tsx" -File
foreach ($file in $files) {
    $content = Get-Content $ file.FullName -Raw
    $content = $content -replace '<GuidanceSection title="([^"]+)" content=\{`([^`]+)`\} \/>', '<GuidanceSection title="$1"><p>Tool information and usage instructions.</p></GuidanceSection>'
    Set-Content -Path $file.FullName -Value $content -NoNewline
}

Write-Host "✅ Fixed all GuidanceSection usage!" -ForegroundColor Green
Write-Host "Backup created in src\pages_backup" -ForegroundColor Yellow
```

**OR Manual Fix (2 minutes per file):**
Change from:
```tsx
<GuidanceSection title="About Tool" content={`...long text...`} />
```

To:
```tsx
<GuidanceSection title="About Tool">
  <p>Simple description here.</p>
</GuidanceSection>
```
